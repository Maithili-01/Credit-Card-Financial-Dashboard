# Credit-Card-Financial-Dashboard
"Power BI dashboard analyzing credit card financial metrics - revenue trends, customer demographics, transaction volumes. SQL data processing with DAX measures for WoW analysis and stakeholder insights ($57M YTD revenue tracked)."



🚀 Project Overview
    Weekly dashboard for real-time credit card analytics.
    Tracks revenue trends, age/income groups, Blue/Silver card transactions (93%), top regions (TX/NY/CA 68%).
    ​YTD: 57.5% activation, 6.06% delinquency rate.

📊 Key Insights (Week 53)
     Revenue +28.8% WoW; males $31M vs females $26M.
    ​ Total transactions $46M, interest $8M.

🛠 Tech Stack
Data: SQL database (CSV imports)

ETL: SQL queries + Power Query

DAX: AgeGroup, CurrentWeekRevenue measures

Visualization: Power BI
